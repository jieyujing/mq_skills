# 掘金量化 (Quant Dig) 技能包

欢迎来到 **Antigravity 的“家”**。本项目为 **掘金量化** 平台提供专属的 Skills 技能包，支持 IDE 直接导入。

## 🚀 快速开始

根据您的使用习惯，您可以将这些技能安装在全局位置或特定项目中。

### 1. Antigravity IDE 集成

参考 Antigravity 官方指南，技能包可以存放在以下两个位置：

#### 🏠 全局位置 (Global)
如果您希望在任何项目里都能使用这些技能，请放置于此：
- **路径**: `~/.gemini/antigravity/skills/`
- **操作**: 将 `mq_skills` 文件夹复制到该目录下。

#### 📂 项目位置 (Workspace)
如果您只希望在当前项目中使用这些技能，请放置于此：
- **路径**: `./.agent/skills/` (位于您的项目根目录)
- **操作**: 如果文件夹不存在请先创建，然后将 `mq_skills` 文件夹复制进去。

### 2. 网页版 Gemini 使用

对于使用网页版 Gemini 的用户，我们提供了专门的压缩包。

- **压缩包**: `mq_skills-gemini.tar.gz`
- **说明**: 该文件专为网页版 Gemini 优化使用。

## 📦 资源说明

- `mq_skills/`: 原始技能文件夹。
- `mq_skills.zip`: 技能文件夹的 ZIP 压缩包。
- `mq_skills-gemini.tar.gz`: 网页版 Gemini 专用包。
